Override and add new features to Sifu in the biz/ directory.

Example directory structure:

biz
|-- includes
|-- media
|   |-- css
|   |-- images
|   |-- js
|   `-- _partitions_
|       `-- foo
|           |-- css
|           |-- images
|           `-- js
|-- modules
|   `-- bar
`-- templates
    |-- bar
    |   `-- languages
    |-- globals
    |   `-- languages
    `-- _partitions_
        `-- foo
            `-- home
                `-- languages
